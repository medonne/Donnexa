# 📖 Sobre este documento

Este documento reúne todas as funcionalidades planejadas para o Donnexa.

O objetivo é organizar a evolução do produto, priorizando entregas de maior valor para o usuário e servindo como guia para o desenvolvimento do projeto.

As funcionalidades estão organizadas por módulos e classificadas por prioridade.

# 🎯 Objetivo do Backlog

O Product Backlog é um documento vivo e será atualizado continuamente conforme novas funcionalidades forem surgindo durante o desenvolvimento do Donnexa.

Cada item representa uma oportunidade de evolução do produto e também uma oportunidade de aprendizado e aplicação de novas tecnologias.
