# 📖 USER STORIES — Donnexa

> Última atualização: Julho de 2026

---

# Sobre este documento

Este documento reúne as histórias de usuário do Donnexa.

Cada história descreve uma necessidade real do usuário e serve como base para o desenvolvimento das funcionalidades do sistema.

As histórias seguem o padrão:

> **Como [tipo de usuário], quero [funcionalidade], para que [benefício].**

---

# 👤 Persona Principal

## Leitor Organizado

- Gosta de acompanhar suas leituras.
- Possui biblioteca física e digital.
- Define metas anuais.
- Busca estatísticas sobre seus hábitos.
- Gosta de desafios pessoais.
- Valoriza organização e simplicidade.

---

# 📚 EPIC 1 — Biblioteca

## US001 — Cadastrar livros

**Como** leitor

**Quero** cadastrar meus livros

**Para que** eu tenha minha biblioteca organizada.

### Critérios de Aceitação

- Deve permitir cadastrar ISBN.
- Deve permitir cadastrar capa.
- Deve permitir informar autor.
- Deve permitir informar editora.
- Deve permitir informar gênero.
- Deve permitir informar número de páginas.

# 📌 Observações

As User Stories serão refinadas continuamente durante o desenvolvimento do Donnexa.

Novas histórias poderão ser adicionadas conforme surgirem novas necessidades, ideias e funcionalidades.

O objetivo deste documento é garantir que todas as funcionalidades do sistema sejam desenvolvidas pensando na experiência do usuário e não apenas na implementação técnica.
