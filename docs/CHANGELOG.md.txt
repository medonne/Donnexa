# Filosofia

O Donnexa é um projeto vivo.

Cada versão representa uma nova etapa de aprendizado.

O histórico deste documento registra não apenas a evolução do software, mas também a evolução das habilidades aplicadas durante seu desenvolvimento.

Cada Sprint concluída representa um novo conhecimento adquirido e incorporado ao projeto.
